# Installation

Get Cartino up and running in your Laravel project in minutes.

## Requirements

- **PHP** 8.2 or higher
- **Laravel** 12.x
- **Database** MySQL 8.0+ or PostgreSQL 13+
- **Node.js** 18+ and npm/pnpm
- **Composer** 2.x

## Quick Installation

Install Cartino via Composer:

```bash
composer require cartinophp/cartino
```

## Setup

Run the installation command:

```bash
php artisan cartino:install
```

This will:
- Publish configuration files
- Run database migrations
- Create default site and channel
- Set up media library
- Publish assets

## Configuration

Configure your database in `.env`:

```ini [.env]
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=cartino
DB_USERNAME=root
DB_PASSWORD=
```

## Frontend Setup

Install JavaScript dependencies:

```bash
npm install
# or
pnpm install
```

Build assets:

```bash
npm run build
# or for development
npm run dev
```

## Verify Installation

Start the development server:

```bash
php artisan serve
```

Visit `http://localhost:8000` to see your Cartino installation.

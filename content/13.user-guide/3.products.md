## Creating a Product

### Basic Information

1. Click **Create Product**
2. Fill in basic details:

![Create Product Form](~/assets/screenshots/create-product.png)

**Required Fields:**
- **Product Name** - Display name (e.g., "Cotton T-Shirt")
- **Slug** - URL-friendly name (auto-generated)
- **Price** - Base price in your store currency
- **Category** - Product category

**Optional Fields:**
- **Description** - Full product description (supports Markdown)
- **SKU** - Stock Keeping Unit (internal reference)
- **Barcode** - UPC, EAN, or ISBN
- **Weight** - For shipping calculations

### Product Images

Upload high-quality product photos:

![Product Images Upload](~/assets/screenshots/product-images.png)

1. Click **Add Images** or drag & drop
2. Upload multiple images (JPG, PNG, WebP)
3. Drag to reorder (first image is the main image)
4. Click **Edit** to crop or add alt text
5. Click **Delete** to remove

**Image Best Practices:**
- Minimum 1000x1000px resolution
- Square aspect ratio (1:1) recommended
- White or neutral background
- Show product from multiple angles
- Max 10MB per image

### Custom Fields (Blueprints)

If your product has a custom blueprint, additional fields appear:

![Blueprint Custom Fields](~/assets/screenshots/product-custom-fields.png)

Example for a T-Shirt:
- **Fabric** - Select: Cotton, Polyester, Blend
- **GSM (Fabric Weight)** - Number input
- **Care Instructions** - Markdown editor
- **Fit Type** - Select: Slim, Regular, Relaxed
- **Size Chart** - Image upload

These fields are defined by your product type's blueprint.

### Variants

Create product variants for different options (size, color, etc.):

![Product Variants](~/assets/screenshots/product-variants.png)

1. Click **Add Variant**
2. Configure options:
   - **Option Name** - e.g., "Size", "Color"
   - **Values** - e.g., "S, M, L, XL"
3. Generate variant combinations
4. Set price, SKU, and stock for each variant

![Variant Editor](~/assets/screenshots/variant-editor.png)

**Variant Fields:**
- **SKU** - Unique identifier
- **Price** - Override base price (optional)
- **Compare at Price** - Original price for discounts
- **Weight** - For shipping
- **Inventory** - Stock quantity

### Inventory

Manage stock levels:

![Inventory Settings](~/assets/screenshots/product-inventory.png)

- **Track Inventory** - Enable stock tracking
- **Quantity** - Current stock level
- **Low Stock Threshold** - Alert when below this number
- **Continue Selling When Out** - Allow backorders

**Inventory Locations:**

If you have multiple warehouses or stores:

![Multi-Location Inventory](~/assets/screenshots/multi-location-inventory.png)

- Set stock per location
- View available inventory by location
- Transfer stock between locations

### SEO

Optimize for search engines:

![Product SEO](~/assets/screenshots/product-seo.png)

- **Meta Title** - Page title (60 characters max)
- **Meta Description** - Search result description (160 characters)
- **URL Handle** - Custom URL slug
- **Social Share Image** - Image for social media previews

**Preview:**
See how your product appears in Google search results.

### Publishing

Control product visibility:

![Product Publishing Options](~/assets/screenshots/product-publishing.png)

- **Status** - Active (visible) or Draft (hidden)
- **Publish Date** - Schedule future publication
- **Sales Channels** - Choose where product appears (Online Store, POS, Facebook, etc.)

### Actions

Bottom of form:

![Product Actions](~/assets/screenshots/product-actions.png)

- **Save as Draft** - Save without publishing
- **Save & Publish** - Make product live
- **Delete Product** - Permanently remove
- **Duplicate Product** - Create a copy

### Collections

Create curated product collections:

![Collections](~/assets/screenshots/collections.png)

1. Go to **Products ‚Üí Collections**
2. Click **Create Collection**
3. Choose collection type:
   - **Manual** - Manually add products
   - **Automatic** - Based on conditions (tags, price, etc.)
4. Add products or set conditions
5. Click **Save**

**Example Automatic Collection:**
- Name: "New Arrivals"
- Condition: Created date is within last 30 days
- Sort by: Newest first

## Import & Export

### Importing Products

Bulk import products from CSV or Excel:

![Product Import](~/assets/screenshots/product-import.png)

1. Go to **Products ‚Üí Import**
2. Download CSV template
3. Fill in product data
4. Upload file
5. Map CSV columns to product fields
6. Preview import
7. Click **Import**

**CSV Format:**
```csv
name,sku,price,category,description,stock
"Cotton T-Shirt",TSH-001,2999,"Clothing","Comfortable cotton tee",100
"Denim Jeans",JNS-001,4999,"Clothing","Classic blue jeans",50
```

### Exporting Products

Export product data:

![Product Export](~/assets/screenshots/product-export.png)

1. Go to **Products ‚Üí Export**
2. Choose format: CSV, Excel, JSON
3. Select fields to export
4. Apply filters (optional)
5. Click **Export**

Use cases:
- Backup product data
- Edit in spreadsheet and re-import
- Share with suppliers
- Analyze inventory

## Best Practices

### Product Names

‚úÖ **Good:**
- "Men's Cotton Crew Neck T-Shirt - Navy Blue"
- "Organic Fair Trade Coffee Beans - 1kg"

‚ùå **Bad:**
- "T-Shirt" (too generic)
- "AWESOME TEE!!!!" (unprofessional)

### Product Descriptions

**Structure:**
1. Brief overview (1-2 sentences)
2. Key features (bullet points)
3. Specifications (size, materials, etc.)
4. Care instructions
5. Shipping information

**Example:**
```markdown
A classic crew neck t-shirt made from 100% organic cotton.

## Features
- Soft, breathable fabric
- Pre-shrunk for lasting fit
- Reinforced shoulder seams
- Available in 5 colors

## Specifications
- Material: 100% Organic Cotton
- Weight: 180 GSM
- Fit: Regular

## Care
Machine wash cold, tumble dry low.
```

### Pricing

- **Use psychological pricing**: $29.99 instead of $30.00
- **Show savings**: Compare at price for discounts
- **Be consistent**: Same product family = similar pricing
- **Include taxes**: Clearly state if prices include VAT/tax

### Images

- **Main image**: Product on white background
- **Detail shots**: Close-ups of features
- **Lifestyle images**: Product in use
- **Size reference**: Product with common objects for scale
- **Multiple angles**: Front, back, side views

## Troubleshooting

### Product Not Appearing on Storefront

Check:
- ‚úÖ Status is "Active"
- ‚úÖ Has at least one image
- ‚úÖ Has inventory (if tracking enabled)
- ‚úÖ Published in "Online Store" sales channel
- ‚úÖ Category is active
- ‚úÖ Publish date is not in the future

### Images Not Uploading

- Check file size (max 10MB)
- Use supported formats (JPG, PNG, WebP)
- Try a different browser
- Check internet connection
- Clear browser cache

### Variants Not Saving

- Ensure each variant has a unique SKU
- Check all required fields are filled
- Try saving with fewer variants first
- Contact support if issue persists

## Processing Orders

### 1. Confirm Payment

For pending orders:

![Confirm Payment](~/assets/screenshots/confirm-payment.png)

1. Verify payment received in your payment gateway
2. Click **Confirm Payment**
3. Order status changes to "Processing"

### 2. Prepare for Shipment

![Prepare Shipment](~/assets/screenshots/prepare-shipment.png)

1. Print packing slip: Click **Print Packing Slip**
2. Pick items from inventory
3. Pack order
4. Click **Mark as Fulfilled** if shipping yourself

### 3. Create Shipping Label

If using integrated shipping:

![Create Shipping Label](~/assets/screenshots/create-shipping-label.png)

1. Click **Create Shipping Label**
2. Choose carrier (USPS, FedEx, UPS, DHL)
3. Select service (Standard, Express, Overnight)
4. Review shipping details
5. Click **Purchase Label**
6. Print label and attach to package

**Integrated Carriers:**
- USPS
- FedEx
- UPS
- DHL Express
- Canada Post
- Royal Mail

### 4. Add Tracking

![Add Tracking](~/assets/screenshots/add-tracking.png)

1. Click **Add Tracking**
2. Enter tracking number
3. Select carrier
4. Click **Save**

Customer receives email with tracking link.

### 5. Mark as Shipped

![Mark as Shipped](~/assets/screenshots/mark-shipped.png)

1. Click **Mark as Shipped**
2. Confirm shipping details
3. Customer receives shipping notification
4. Order status changes to "Shipped"

## Fulfillment

### Manual Fulfillment

For digital products or in-store pickup:

![Manual Fulfillment](~/assets/screenshots/manual-fulfillment.png)

1. Click **Fulfill Order**
2. Select items to fulfill (can be partial)
3. Click **Fulfill**
4. Order status updates

### Partial Fulfillment

Ship items as they become available:

![Partial Fulfillment](~/assets/screenshots/partial-fulfillment.png)

1. Select items ready to ship
2. Click **Fulfill Selected**
3. Add tracking for this shipment
4. Repeat for remaining items

**Example:**
- Order: 2x T-Shirts, 1x Jacket
- Fulfillment 1: Ship 2x T-Shirts
- Fulfillment 2: Ship 1x Jacket (when back in stock)

### Split Shipments

Items shipped from different locations:

![Split Shipments](~/assets/screenshots/split-shipments.png)

Each fulfillment gets:
- Separate tracking number
- Individual notification email
- Own packing slip

## Refunds & Returns

### Full Refund

![Full Refund](~/assets/screenshots/full-refund.png)

1. Click **Refund**
2. Select all items
3. Choose refund amount (full or partial)
4. Select refund method:
   - **Original payment method** (default)
   - **Store credit**
   - **Manual refund** (outside Cartino)
5. Add refund reason (optional)
6. Click **Process Refund**

Refund appears in customer's account within 5-10 business days.

### Partial Refund

Refund specific items or amounts:

![Partial Refund](~/assets/screenshots/partial-refund.png)

1. Click **Refund**
2. Select items to refund
3. Adjust quantities if needed
4. Modify refund amount (e.g., for damaged items)
5. Choose whether to restock items
6. Click **Process Refund**

### Restocking

![Restock Options](~/assets/screenshots/restock-options.png)

When processing refunds:
- ‚úÖ **Restock items** - Add items back to inventory
- ‚ùå **Don't restock** - For damaged/defective items

### Refund Shipping

![Refund Shipping](~/assets/screenshots/refund-shipping.png)

Include shipping in refund:
- Check **Refund shipping**
- Specify amount to refund
- Add reason in notes

### Store Credit

Issue store credit instead of refund:

![Store Credit](~/assets/screenshots/store-credit.png)

1. Select **Store Credit** as refund method
2. Credit appears in customer account
3. Customer can use on future purchases
4. Never expires (configurable)

## Fraudulent Orders

Protect against fraud:

![Fraud Analysis](~/assets/screenshots/fraud-analysis.png)

**Risk Indicators:**
- Billing vs shipping address mismatch
- Multiple failed payment attempts
- Suspicious email domain
- High-value first-time order
- IP address location mismatch

**Actions:**
- Review order manually
- Contact customer
- Cancel and refund
- Block customer (if confirmed fraud)

## Troubleshooting

### Payment Not Showing

1. Check payment gateway dashboard
2. Verify webhook configuration
3. Manually mark as paid if confirmed
4. Contact support if recurring

### Can't Create Shipping Label

- Verify shipping address is complete
- Check shipping carrier integration
- Ensure package weight is entered
- Confirm carrier account is active

### Customer Didn't Receive Email

- Check spam folder
- Verify email address is correct
- Resend notification manually
- Check email logs in **Settings ‚Üí Email**

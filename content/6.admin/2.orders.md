# Order Management

Process and manage customer orders from creation to fulfillment in Cartino's admin panel.

[TOC]

## Orders Overview

Navigate to **Orders → All Orders** to view order list.

### Order List View

The order list displays:

- Order number
- Customer name and email
- Order status
- Payment status
- Total amount
- Date and time
- Quick actions

### Order Statuses

| Status | Description | Actions Available |
|--------|-------------|-------------------|
| **Pending** | Awaiting payment | Cancel, Process payment |
| **Processing** | Payment received, preparing | Fulfill, Cancel |
| **Fulfilled** | Shipped to customer | Mark as delivered, Refund |
| **Delivered** | Received by customer | Refund |
| **Cancelled** | Order cancelled | View only |
| **Refunded** | Money returned | View only |

### Filtering Orders

Filter by:

- **Status**: Any status combination
- **Payment Status**: Paid, Pending, Failed
- **Date Range**: Today, Week, Month, Custom
- **Customer**: Search by name or email
- **Amount**: Min/Max range
- **Channel**: Online, POS, Mobile, etc.

### Bulk Actions

Select multiple orders:

- Export to CSV/Excel
- Print packing slips
- Print invoices
- Update status
- Send notifications

## Creating Orders Manually

**Orders → Create Order**

Use for phone orders, admin sales, or manual entry.

### Step 1: Select Customer

- Search existing customer
- Or create new customer
- Enter email and basic info

### Step 2: Add Products

1. Search products
2. Select variant
3. Set quantity
4. Add to order
5. Repeat for all items

**Edit Line Items:**

- Adjust quantity
- Override price
- Add custom discount
- Add notes

### Step 3: Add Shipping

1. Enter shipping address
2. Select shipping method
3. Set shipping cost
4. Add tracking (optional)

### Step 4: Payment

- Select payment method
- Mark as paid (if payment received)
- Add payment reference
- Or leave unpaid (send invoice)

### Step 5: Review & Create

- Review all details
- Add order notes (internal)
- Add customer notes (visible to customer)
- Create order

**Options:**

- Send confirmation email
- Create invoice
- Reserve inventory
- Calculate taxes automatically

## Processing Orders

### Mark as Processing

**Order Details → Mark as Processing**

When payment is received:

1. Verify payment
2. Update status to Processing
3. Reserve inventory
4. Send confirmation email
5. Print packing slip

### Editing Orders

Before fulfillment, you can:

- Add or remove items
- Adjust quantities
- Change prices
- Apply discounts
- Update addresses
- Modify shipping method

**After fulfillment:**

- Limited changes allowed
- Must create refund for major changes

## Order Fulfillment

### Fulfill Order

**Order Details → Fulfill Order**

### Single-Location Fulfillment

1. Review items to fulfill
2. Select location (if multi-location)
3. Add tracking information:
   - Carrier (UPS, FedEx, USPS, etc.)
   - Tracking number
   - Tracking URL (auto-generated)
4. Notify customer (checkbox)
5. Click Fulfill

### Multi-Location Fulfillment

For orders with items from multiple locations:

**Partial Fulfillment:**

1. Select items from Location A
2. Add tracking for shipment A
3. Fulfill selected items
4. Repeat for Location B
5. Multiple shipments created

**Order Status:**

- Partially Fulfilled (until all items shipped)
- Fulfilled (when all items shipped)

### Packing Slips

**Print Packing Slip:**

- Order Details → Print Packing Slip
- Lists all items with quantities
- Includes addresses
- Shows special instructions
- Barcode for easy scanning

**Customize Packing Slips:**

Settings → Packing Slips

- Add logo
- Custom messages
- Terms and conditions
- Return instructions

### Tracking Updates

**Automatic Tracking:**

- Integration with carriers (UPS, FedEx, etc.)
- Automatic status updates
- Email notifications to customer
- Tracking page on your site

**Manual Updates:**

- Add tracking number manually
- Update delivery status
- Add delivery notes

## Returns & Refunds

### Process Return

**Order Details → Process Return**

1. Select items to return
2. Choose reason:
   - Customer changed mind
   - Product defective
   - Wrong item sent
   - Size/fit issues
   - Other
3. Restock inventory (yes/no)
4. Issue refund or store credit
5. Send RMA email to customer

### Issue Refund

**Order Details → Refund**

**Full Refund:**

- Select all items
- Refund shipping (optional)
- Restock items
- Process refund

**Partial Refund:**

- Select specific items
- Or enter custom amount
- Choose refund reason
- Add notes
- Process refund

**Refund Methods:**

- Original payment method
- Store credit
- Manual refund (mark as refunded)

### Refund to Payment Method

Automatic refund if payment gateway supports:

- Stripe
- PayPal
- Square
- Braintree

**Processing time:**

- Credit card: 5-10 business days
- PayPal: 3-5 business days
- Debit card: 5-10 business days

### Store Credit

Issue store credit instead:

- Faster for customer
- Encourages repeat purchase
- No transaction fees

**Customer receives:**

- Store credit balance
- Code to use at checkout
- Expiration date (optional)

### Restocking

When refunding, choose to restock:

- **Yes**: Add quantity back to inventory
- **No**: Don't adjust inventory (damaged items)
- **Different Location**: Restock to different warehouse

## Abandoned Carts

### View Abandoned Carts

**Orders → Abandoned Carts**

Shows carts with:

- Items added
- No order completed
- Created within recovery window (default 24 hours)

**Cart Information:**

- Customer email (if logged in)
- Items in cart
- Cart value
- Time abandoned
- Recovery status

### Recovery Emails

**Automatic Recovery:**

Settings → Abandoned Carts → Enable Recovery

- Send after X hours (default: 1 hour)
- Series of emails (3 recommended)
- Include cart contents
- Add discount code (optional)

**Email Schedule:**

- Email 1: 1 hour after abandonment
- Email 2: 24 hours after
- Email 3: 72 hours after

### Manual Recovery

Select abandoned cart:

- Send recovery email manually
- Create order on behalf of customer
- Contact customer directly

### Recovery Analytics

View metrics:

- Abandonment rate
- Recovery rate
- Revenue recovered
- Best performing emails

## Tips & Best Practices

### Processing Efficiency

- Process orders same-day when possible
- Use bulk actions for multiple orders
- Print packing slips in batches
- Set up automatic fulfillment for digital goods

### Customer Communication

- Always send tracking information
- Update customers on delays
- Be proactive with issue resolution
- Use automated emails consistently

### Inventory Management

- Fulfill orders from nearest location
- Monitor stock levels during fulfillment
- Set low-stock alerts
- Regular inventory audits

### Fraud Prevention

- Verify large or unusual orders
- Check for address mismatches
- Monitor for duplicate orders
- Use AVS (Address Verification)
- Require CVV for card payments

### Returns Processing

- Quick response to return requests
- Clear return policy
- Easy return process
- Process refunds promptly
- Track return reasons for insights

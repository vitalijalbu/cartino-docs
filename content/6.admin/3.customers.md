## Creating Customers

**Customers → Add Customer**

### Manual Creation

For phone orders or in-person registration:

1. **Basic Information**
   - Email address (required, unique)
   - First name
   - Last name
   - Phone number

2. **Account Settings**
   - Send welcome email (checkbox)
   - Assign to customer group
   - Set password (or send reset link)
   - Mark email as verified

3. **Additional Details**
   - Date of birth
   - Tax ID (for B2B)
   - Company name (B2B)
   - Notes

4. **Address** (optional)
   - Add shipping address
   - Add billing address

5. **Marketing**
   - Accepts marketing emails
   - Accepts SMS marketing
   - Communication preferences

### Bulk Import

**Customers → Import**

Import from CSV/Excel:

1. Download template
2. Fill customer data
3. Upload file
4. Map columns
5. Validate and import

**CSV Format:**

```csv
email,first_name,last_name,phone,customer_group,accepts_marketing
john@example.com,John,Doe,+1234567890,Retail,yes
jane@example.com,Jane,Smith,+0987654321,Wholesale,no
```

## Editing Customers

### Edit Customer Information

**Customer Profile → Edit**

Update any field:

- Personal information
- Contact details
- Customer group
- Account status
- Tax exemptions

### Reset Password

**Customer Profile → Reset Password**

Options:

- Generate and send reset link
- Set password manually
- Force password change on next login

### Account Status

Change customer account status:

- **Active**: Normal account
- **Inactive**: Disabled login (soft)
- **Banned**: Blocked (fraud, abuse)

**Effects of Status:**

| Status | Can Login | Can Order | Receives Emails |
|--------|-----------|-----------|-----------------|
| Active | Yes | Yes | Yes |
| Inactive | No | No | No |
| Banned | No | No | No |

## Customer Groups

Create customer segments for special pricing and promotions.

### View Customer Groups

**Customers → Customer Groups**

Default groups:

- **Retail**: Regular customers
- **Wholesale**: Bulk buyers
- **VIP**: High-value customers
- **Staff**: Employee discount

### Create Customer Group

**Add Customer Group**

1. **Group Name**: "Wholesale"
2. **Description**: Internal description
3. **Discount**: Percentage or fixed amount
4. **Pricing Rules**: Custom pricing tier
5. **Auto-Assign Rules**: (optional)
   - Total spent > $1000
   - Total orders > 10
   - Tags contain "wholesale"

### Group Benefits

Configure benefits per group:

**Pricing:**

- Percentage discount (e.g., 15% off)
- Special pricing tier
- Volume discounts
- Free shipping threshold

**Access:**

- Exclusive products
- Early access to sales
- Private collections
- Special payment terms

**Experience:**

- Priority support
- Dedicated account manager
- Custom catalogs
- Net payment terms (B2B)

### Assign Customers to Groups

**Manual Assignment:**

Customer Profile → Customer Group → Select

**Automatic Assignment:**

Create rules in group settings:

```text
Auto-assign to VIP when:
- Total spent > $5,000
- Total orders > 20
- Account age > 1 year
```

**Bulk Assignment:**

1. Select multiple customers
2. Bulk Actions → Assign to Group
3. Choose group
4. Apply

## Loyalty Program

### Fidelity Cards

**Customers → Loyalty Cards**

Issue loyalty cards to customers for points and rewards.

### Create Fidelity Card

**Customer Profile → Issue Fidelity Card**

1. Card number (auto-generated)
2. Starting points (optional)
3. Tier/Level
4. Expiration date (optional)
5. Notes

### Points System

**Settings → Loyalty → Points**

Configure earning rules:

**Earn Points:**

```text
Earn 1 point per $1 spent
Earn 2x points on birthdays
Earn 100 points on sign-up
Earn 50 points per referral
```

**Redeem Points:**

```text
100 points = $10 discount
500 points = $50 discount
1000 points = $120 discount (bonus value)
```

### Loyalty Tiers

Create tiered rewards:

| Tier | Requirements | Benefits |
|------|-------------|----------|
| **Bronze** | 0-499 points | 1x points, 5% discount |
| **Silver** | 500-1999 points | 1.5x points, 10% discount |
| **Gold** | 2000+ points | 2x points, 15% discount |

**Tier Benefits:**

- Point multipliers
- Exclusive discounts
- Free shipping
- Early access
- Birthday rewards

### Points Transactions

**Customer Profile → Loyalty → Transactions**

View all point activity:

- Points earned (orders)
- Points redeemed (rewards)
- Points adjusted (manual)
- Points expired
- Running balance

**Manual Adjustment:**

- Add points (goodwill, promotion)
- Deduct points (correction)
- Add reason/note

**Analytics → RFM Analysis**

- **Recency**: Last purchase date
- **Frequency**: Number of orders

**Segments Generated:**

- Loyal customers
- Potential loyalists
- At-risk customers
- Lost customers

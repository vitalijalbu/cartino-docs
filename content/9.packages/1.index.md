# Packages & Libraries

Cartino leverages powerful Laravel packages to provide robust functionality out of the box. This section covers the key packages integrated into Cartino and how to use them effectively.

## Core Packages

### Spatie Ecosystem

Cartino heavily relies on the excellent **Spatie** packages:

- **[Media Library](/packages/media-library)** - Manage product images, documents, and other media files
- **[Permissions](/packages/permissions)** - Role-based access control for users and customers
- **[Query Builder](/packages/query-builder)** - Advanced filtering and sorting for API endpoints

### Laravel Official

- **[Sanctum](/packages/sanctum-advanced)** - API authentication with tokens
- **[Notifications](/packages/notifications)** - Multi-channel notification system
- **[Mail](/packages/mailable)** - Customizable email templates

## Why These Packages?

Each package is battle-tested, maintained by the community, and follows Laravel best practices. They integrate seamlessly with Cartino's architecture while remaining flexible for customization.

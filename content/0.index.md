## Powerful Blueprints

::split-section
---
image: /images/features/blueprints.svg
imagePosition: right
---

#title
Organize Fields to Structure Your Content

#description
Inspired by Statamic CMS, Blueprints let you define custom fields for products, orders, customers, and any data type. No database migrations needed‚Äîjust YAML.

#content
```yaml
# blueprints/products/tshirt.yaml
title: T-Shirt Product
sections:
  details:
    fields:
      - handle: fabric
        field:
          type: select
          options:
            - Cotton
            - Polyester
            - Blend
      - handle: gsm
        field:
          type: number
          display: Fabric Weight (GSM)
      - handle: care_instructions
        field:
          type: markdown
```

# Installation

Add Cartino UI to your Vue or Nuxt project.

## Requirements

- Vue 3.3+ or Nuxt 3.10+
- Node.js 18+

## Nuxt Installation

Install the Nuxt module:

:::u-tabs
---
items:
  - label: pnpm
    slot: pnpm
  - label: npm
    slot: npm
  - label: yarn
    slot: yarn
---

#pnpm
```bash
pnpm add @cartino/ui
```

#npm
```bash
npm install @cartino/ui
```

#yarn
```bash
yarn add @cartino/ui
```
:::

Add to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@cartino/ui'],
  
  cartino: {
    theme: {
      colors: {
        primary: 'blue'
      }
    }
  }
})
```

## Vue Installation

For standalone Vue projects:

```bash
pnpm add @cartino/ui
```

Import in your `main.ts`:

```ts [src/main.ts]
import { createApp } from 'vue'
import CartinoUI from '@cartino/ui'
import '@cartino/ui/dist/style.css'
import App from './App.vue'

const app = createApp(App)
app.use(CartinoUI, {
  theme: {
    colors: {
      primary: 'blue'
    }
  }
})
app.mount('#app')
```

## Tailwind Configuration

Cartino UI requires Tailwind CSS. Add to your `tailwind.config.js`:

```js [tailwind.config.js]
export default {
  content: [
    './components/**/*.{vue,js,ts}',
    './pages/**/*.vue',
    './node_modules/@cartino/ui/**/*.{vue,js,ts}'
  ],
  theme: {
    extend: {}
  },
  plugins: []
}
```

## TypeScript

Types are included. For better IntelliSense, add to `tsconfig.json`:

```json [tsconfig.json]
{
  "compilerOptions": {
    "types": ["@cartino/ui"]
  }
}
```

## Verify Installation

Test with a simple component:

```vue
<template>
  <ui-button>Hello Cartino!</ui-button>
</template>
```

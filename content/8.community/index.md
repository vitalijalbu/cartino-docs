### Contributing Code

1. **Fork the Repository**

   ```bash
   git clone https://github.com/cartinophp/cartino.git
   cd cartino
   ```

2. **Create a Branch**

   ```bash
   git checkout -b feature/my-new-feature
   ```

3. **Make Your Changes**

   - Follow PSR-12 coding standards
   - Write tests for new features
   - Update documentation

4. **Run Tests**

   ```bash
   composer test
   ```

5. **Submit Pull Request**

   - Clear description of changes
   - Link related issues
   - Add screenshots if UI changes

**Contribution Guidelines:**

- [Contributing Guide](https://github.com/cartinophp/cartino/blob/main/CONTRIBUTING.md)
- [Code of Conduct](https://github.com/cartinophp/cartino/blob/main/CODE_OF_CONDUCT.md)

## Getting Support

### Before Asking

Help us help you:

1. **Search First**
   - Check documentation
   - Search GitHub issues
   - Browse Discord history
   - Google your error

2. **Prepare Information**
   - Cartino version
   - PHP version
   - Laravel version
   - Error messages
   - Steps to reproduce

3. **Create Minimal Example**
   - Isolate the problem
   - Remove unnecessary code
   - Test with fresh install

### How to Ask

**Good Question:**

```text
Title: Cart total calculation incorrect with multiple discounts

Environment:
- Cartino: v2.1.0
- PHP: 8.2
- Laravel: 11.0

Problem:
When applying multiple discount codes, the cart total shows
incorrect amount. Only the first discount is applied.

Steps to Reproduce:
1. Add product to cart
2. Apply discount code "SAVE10"
3. Apply discount code "WELCOME5"
4. Expected: Both discounts applied
5. Actual: Only "SAVE10" applied

Code:
[Paste relevant code]

Error Message:
[Paste error if any]
```

**Bad Question:**

```text
Cart not working help!!!
```
